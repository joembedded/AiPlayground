# Funktion: scherz

## Zweck
Liefert einen Scherz / Streich / Witz / Anekdote basierend auf einem Argument (z.B. Person, Situation, Ort, Gegenstand).
Diese Funktion wird gewählt, wenn der Nutzer nach einem Witz, einer Andekdote, einem Streich oder “etwas Lustigem” fragt.

## Wann verwenden
- Der Nutzer möchte einen Scherz / Streich / Witz / Lustiges / Andekdote.
- Formulierungen wie: "mach einen Scherz", "hecke einen Streich aus", "witzige Idee", "kleine Geschichte".

## Eingabe (arguments)
- ziel (string, required): Für wen/was ist der Scherz? (z.B. "Kollege Tom", "meine Schwester", "Büro")
- kontext (string, optional): Rahmen/Situation (z.B. "im Büro", "Geburtstag", "Meeting")

## Ausgabe
- String

## Beispiele
- Nutzer: "Mache einen Witz über Hans im Büro"
  -> tool_name: "scherz"
  -> arguments: { "ziel": "Hans", "kontext": "Büro"}

- Nutzer: "Erzähle eine Anekdote über meine Schwester"
  -> tool_name: "scherz"
  -> arguments: { "ziel": "Schwester" }

# Funktion: rechnung

## Zweck
Führe eine einfache Berechnung oder Zähl-/Auswertelogik basierend auf einem Argument aus.

## Wann verwenden
- Der Nutzer möchte rechnen, zählen, berechnen oder auswerten.
- Formulierungen wie: "berechne", "rechne", "wie viele", "zähle", "ermittle", "Summe", "Differenz".
- Beispiel: Buchstaben in einem Namen zählen.

## Eingabe (arguments)
- operation (string, required): Art der Berechnung
  - Erlaubte Werte (Beispiele): "buchstaben_zaehlen", "quersumme", "laenge", "summe"
- input (string, required): Eingabewert für die Berechnung (z.B. Name, Text oder Zahl als String)

## Ausgabe
- Ein String oder eine Zahl.

## Beispiele
- Nutzer: "Zähle die Buchstaben in 'Haus'."
  -> tool_name: "rechnung"
  -> arguments: { "operation": "buchstaben_zaehlen", "input": "Haus" }

- Nutzer: "Wievel ist 3 + 3?"
  -> tool_name: "rechnung"
  -> arguments: { "operation": "summe", "input": "3 + 3"}
